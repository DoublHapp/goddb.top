# 阅读 Vue 渲染器的三个切入点

大型框架源码并不适合从第一行顺序阅读。理解 Vue 渲染器，可以先建立三个相互连接的观察点。

## VNode 描述什么

VNode 不只是 DOM 的抽象。它同时携带节点类型、子节点形态和编译阶段提供的优化信息，是编译器与运行时之间的协议。

## patch 如何缩小问题

patch 的核心任务是根据新旧节点关系选择最小更新路径。先关注元素、组件和 Fragment 三条主分支，再深入具体优化。

## 标志位为什么重要

shapeFlag 描述节点的大类，patchFlag 描述动态内容。前者帮助运行时快速分派，后者让编译信息直接参与更新。

带着问题阅读调用链，比记住每个函数更有效。
